# Guessing Game app

## Description

Guessing Game app is a React Native app that offers a game to be played between AI and human.

## Purpose

The goal of this app is to get familiar with the React Native framework and compare it to Flutter for cross-platform
development.

## Topics covered

- 👨‍🎨 Styling
- 🏛️ Layout
- 🔤 User input
- 🗃️ State Management
- 📲 Platform specific
- 🌆 Responsive design
